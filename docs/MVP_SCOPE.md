# MVP 边界定义 (MVP Scope)

## 第一版一定做

```
✅ 图片上传（支持拖拽）

✅ 图片裁剪（固定比例）

✅ 实时像素化预览

✅ 预设调色板（Perler 官方 216 色）

✅ 自定义调色板

✅ 颜色映射（LAB + ΔE2000）

✅ 网格绘制

✅ 拼板自动分割

✅ PNG 导出（单板/拼接）

✅ PDF 导出（图纸 + 材料清单）

✅ 颜色统计

✅ 本地存储

✅ 快捷键支持（Ctrl+Z/Y, Ctrl+S）
```

---

## 第一版绝对不做

```
❌ AI 生成图片

❌ 登录/注册

❌ 社区/论坛

❌ 云同步

❌ 模板商城

❌ 多人协作

❌ 插件市场

❌ 移动端/平板适配

❌ 付费功能

❌ SaaS
```

---

## 边界说明

如果有人要求实现以上"绝对不做"的功能，答案是：**第一版不考虑，后续看用户反馈**。

这不是技术限制，是产品策略。
