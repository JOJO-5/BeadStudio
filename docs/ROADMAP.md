## v0.3.0 — 图片处理

**目标**：实现完整的图片上传到像素化预览流程。

### 功能

- [ ] 图片上传（拖拽 + 点击）
- [ ] 图片裁剪（react-easy-crop）
- [ ] 图片缩放（Pica 高质量缩放）
- [ ] 像素化预览（实时预览效果）
- [ ] Worker 异步处理

### 交付物

- 图片上传 → 裁剪 → 缩放 → 像素化完整流程
- 所有图片处理在 Worker 中完成，不阻塞 UI

---

## v0.4.0 — 颜色引擎

**目标**：实现专业的颜色匹配能力。

### 功能

- [ ] 预设调色板（Perler 官方色）
- [ ] 自定义调色板
- [ ] LAB + ΔE2000 颜色匹配
- [ ] 抖动算法（Floyd-Steinberg / Atkinson / Sierra）
- [ ] 颜色替换
- [ ] 颜色统计（数量 / 百分比）

### 技术指标

- ΔE 阈值可配置（默认 < 5.0）
- 支持 20+ 预设调色板
- 抖动算法可切换

---

## v0.5.0 — 网格与导出

**目标**：实现完整的拼板和导出能力。

### 功能

- [ ] 网格绘制（不同 bead 尺寸）
- [ ] 自动拼板（计算板子数量和排列）
- [ ] PNG 导出（单板 / 拼接图）
- [ ] PDF 导出（图纸 + 颜色编号 + 材料清单）
- [ ] 网格坐标标注
- [ ] 材料采购清单

### 导出格式

| 格式 | 内容 |
|------|------|
| PNG 单板 | 每块板的独立图片 |
| PNG 全图 | 所有板子拼接的大图 |
| PDF 图纸 | 含坐标、颜色编号的施工图 |
| PDF 清单 | 材料数量统计表 |
