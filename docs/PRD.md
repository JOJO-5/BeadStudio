# 产品需求文档 (PRD)

## 产品定位

BeadStudio（拼豆大师）是最简单、最快速、最好用的浏览器端拼豆设计工具。

**不是** Photoshop、Photopea、Figma 或任何通用图像编辑器。

**只是** 一款专注于拼豆创意的设计工具。

---

## 核心价值

帮助用户快速完成：

```
图片 → 像素化设计 → 拼板图纸 → 导出生产
```

---

## 用户故事

### 作为玩家，我想...
上传一张图片，快速看到它变成拼豆效果

### 作为玩家，我想...
选择一个预设调色板（Perler/Hama/Artkal），让颜色自动映射

### 作为玩家，我想...
导出清晰的拼板图纸，包含坐标标注和材料清单

### 作为玩家，我想...
不需要注册账号，打开浏览器就能用

---

## 产品原则

**任何功能新增必须回答：它是否帮助用户更快完成拼豆设计？**

如果否，则拒绝开发。

---

## 竞品分析

| 竞品 | 缺点 | BeadStudio 优势 |
|------|------|----------------|
| Perler Beads Studio | 仅限 Perler，功能单一 | 多品牌调色板 |
| PixelIt | 无实时预览，无导出 | 完整导出 (PNG/PDF) |
| Perler PhotoPea Plugin | 需要 Photopea，操作复杂 | 独立工具，零学习成本 |
| 手工论坛教程 | 纯图文，效率低 | 所见即所得 |

---

## 功能优先级

### P0（必须做）

- 图片上传（拖拽/点击）
- 裁剪
- 实时像素化预览
- 调色板选择
- 网格绘制
- PNG 导出
- PDF 导出（含图纸+清单）

### P1（下个版本）

- 多品牌调色板（Hama/Artkal）
- 抖动算法切换
- 颜色替换

### P2（未来版本）

- 模板市场
- 社区分享
- 云同步

### P3（绝对不做）

- AI 生成图片
- 登录系统
- SaaS 收费
- 多人协作
- 插件市场
